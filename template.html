Chaosbackend

This is a backend meant to be used with varnish to test different scenarios.

/error
Returns a 500 internal server error
/error?status=404
Returns a 404 error
/error?status=404&sleep=5000
Returns a 404 error after 5 seconds
/reset
Close connection like a champ!
/slow?sleep=1000&sleepBetweenBytes=100
Returns a 200 but waits 1 second before start delivering, then wait 0.1 seconds between bytes
